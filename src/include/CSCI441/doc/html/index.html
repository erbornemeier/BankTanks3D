CSCI441 Library Documentation

This library is intended to be used with OpenGL for CSCI441 at the Colorado School of Mines.

When building, the library must be compiled and linked against OpenGL and glm. Headers thatend in *3.hpp are built for OpenGL 3.0+ and additionally depend upon GLEW.
